Les Positions

La propriété css position permet de controler le positionnement d'un element dans le flux de la page.

position : static;

La position par defaut de tout element html, positionné dans le flux de la page

position: relative;

L'élément est placé dans le flux normal puis déplacé par rapport a son emplacement d'origine, pour cela nous utilisons top, right, left, bottom

Element en relative

Position : absolute;

L'élément est retiré du flux normal et positionné par rapport à son parent positionné (relative, absolute, fixed ou en sticky) le plus proche, si aucun parent n'est positionné, il se placera par rapport à l'élément html
